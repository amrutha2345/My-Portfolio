Library Management System

Description

The Library Management System is a simple yet functional Python application for managing book inventories and user memberships. It demonstrates the practical use of file handling, data structures, and object-oriented programming in Python.

Key Features

  • Add, update, and delete books in the catalog
  • Register new members and maintain member records
  • Issue and return books with due date tracking
  • View available books and issued book details
  • Console-based user interface for easy use

VOTER'S PRIDE

Description

The Online Voting System is a secure Python-based application that allows registered users to cast their votes digitally. This project demonstrates my skills in implementing authentication, data validation, and managing voting records.

Key Features

  • User registration and login authentication
  • Candidate registration and display
  • Secure voting process (one vote per user)
  • Vote counting and result declaration
  • Admin functionality for managing users and candidates
  • Console-based user interface
